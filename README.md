# EduFlex — B2B2C платформа корпоративного обучения

## Elevator Pitch

EduFlex — это B2B2C-платформа для корпоративного обучения, которая помогает малому и среднему бизнесу быстро прокачивать сотрудников в digital-навыках.
В отличие от классических LMS, EduFlex предлагает персонализированные карьерные треки, адаптивное обучение и аналитику эффективности, ориентированную не только на компанию, но и на конечного пользователя — сотрудника.
Это позволяет бизнесу быстрее закрывать skill-gap, а сотрудникам — расти внутри компании без смены работы.
